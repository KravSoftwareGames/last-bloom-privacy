Privacy Policy

Last updated: May 1, 2026

This Privacy Policy applies only to the mobile game Last Bloom, developed and published by KravSoftware Games.

1. Overview

Last Bloom is a mobile match-3 puzzle game. This Privacy Policy explains how the game handles user data and privacy.

2. Data Collection

The current test version of Last Bloom does not require account registration and does not directly collect personal information from users.

The current version of Last Bloom does not use:

  • user accounts;
  • advertising SDKs;
  • analytics SDKs;
  • location tracking;
  • access to contacts;
  • access to microphone or camera;
  • cloud save services;
  • in-app purchases.

3. Data Sharing

The current version of Last Bloom does not share personal user data with third parties.

4. Data Security

KravSoftware Games aims to protect user privacy and only use data that is necessary for the game to function.

5. Data Retention and Deletion

The current version of Last Bloom does not create user accounts and does not store personal account data. If future versions include accounts or online features, users will be provided with a way to request data deletion where required.

6. Children’s Privacy

Last Bloom is not specifically directed to children. If this changes in the future, this Privacy Policy and app settings will be updated accordingly.

7. Future Changes

If future versions of Last Bloom include analytics, advertising, cloud saves, in-app purchases, online features, or account systems, this Privacy Policy will be updated before those features are released.
